# Controle de Gastos

Controle de Gastos é uma aplicação web simples para gerenciar suas despesas pessoais de forma prática e eficiente.  
O projeto foi desenvolvido com **HTML, CSS e JavaScript**, utilizando **localStorage** para salvar os dados no navegador.

## 🔗 Acesse o site

Você pode acessar a aplicação hospedada no GitHub Pages:  
[https://danieeltrindade.github.io/controle-de-gastos/](https://danieeltrindade.github.io/controle-de-gastos/)

## 📝 Funcionalidades

- Cadastro de gastos com:
  - Descrição
  - Valor
  - Categoria (Alimentação, Transporte, Lazer, Outros)
- Listagem de todos os gastos cadastrados
- Cálculo automático do total gasto
- Exclusão de gastos individuais
- Mensagem de confirmação ao adicionar gastos
- Persistência de dados usando **localStorage**
